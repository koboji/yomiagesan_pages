はじめに

「読み上げさん」（以下「本アプリ」）は、ユーザーのプライバシーを尊重し、個人情報の保護に努めます。本プライバシーポリシーは、本アプリにおける情報の取り扱いについて説明します。

収集しない情報

本アプリは以下の情報を収集しません：

  • 個人を特定できる情報（氏名、メールアドレス等）
  • 位置情報
  • アクセス解析やトラッキングデータ
  • 広告識別子

端末内に保存される情報

以下の情報はすべてユーザーの端末内にのみ保存され、外部に送信されることはありません：

  • ユーザーが入力・読み込んだテキストデータ
  • フォルダ構成
  • アプリの設定情報（音声エンジン、読み上げ速度、外観設定等）
  • 読み上げ位置の記録

Google Cloud Text-to-Speech API について

本アプリでは、オプション機能としてGoogle Cloud Text-to-Speech APIを利用できます。この機能を有効にした場合：

  • 読み上げ対象のテキストがGoogleのサーバーに送信されます
  • ユーザー自身が取得・入力したAPIキーが認証に使用されます
  • APIキーは端末のKeychain（安全な保存領域）に保存されます

この機能はオプションであり、有効にしない場合、テキストデータが外部に送信されることは一切ありません。Googleでのデータの取り扱いについては、Google Cloud のプライバシーポリシーをご確認ください。

カメラ・フォトライブラリ

  • OCR（文字認識）機能のために、カメラまたはフォトライブラリへのアクセスを要求する場合があります
  • 撮影・選択された画像は端末内で文字認識処理に使用された後、アプリ内に保存されることはありません
  • 画像が外部サーバーに送信されることはありません

データの保存場所

すべてのユーザーデータは端末内に保存されます。iCloudやその他のクラウドサービスへの同期は行いません。

第三者への提供

本アプリは、Google Cloud TTS利用時を除き、ユーザーのデータを第三者に提供、販売、共有することはありません。

お子様のプライバシー

本アプリは特にお子様を対象としたものではありませんが、個人情報を収集しないため、あらゆる年齢のユーザーが安心してご利用いただけます。

プライバシーポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更がある場合は、アプリ内でお知らせします。
